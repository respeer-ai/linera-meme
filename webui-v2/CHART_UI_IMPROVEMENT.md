# 图表 UI 改进方案

## 📊 当前问题分析

### 现有图表的不足

1. **工具栏简陋** - 只有时间周期选择器，缺少其他控制
2. **信息显示混乱** - OHLCV 和 MA 信息叠加在图表上，遮挡视线
3. **缺少图表类型切换** - 只有蜡烛图
4. **缺少指标控制** - MA 固定显示，无法配置
5. **视觉效果一般** - 工具栏样式简单，不够专业

## 🎯 改进目标

### 参考 Raydium/TradingView 的专业图表

- 完整的工具栏（图表类型、指标、设置）
- 清晰的信息显示（不遮挡图表）
- 可配置的指标（MA 周期可调）
- 更多图表类型（折线图、面积图）
- 专业的视觉效果

## 🔧 具体改进内容

### 1. 增强工具栏 (ChartToolbar.vue)

```
┌────────────────────────────────────────────────────────────┐
│ [蜡烛图▼] [指标▼] │ 1m 5m 15m 1h 4h 1d │ [全屏] [设置] │
└────────────────────────────────────────────────────────────┘
```

**新增功能：**

- 图表类型选择器（蜡烛图/折线图/面积图）
- 指标选择器（MA/EMA/BOLL 可选）
- 全屏按钮
- 设置按钮（配置 MA 周期、颜色等）

### 2. 优化信息显示 (ChartOverlay.vue)

**当前问题：** 信息叠加在图表上，遮挡 K 线

**改进方案：**

- 信息显示在工具栏下方独立区域
- 使用更紧凑的布局
- 悬停时高亮显示
- 支持折叠/展开

### 3. 图表类型切换

**新增类型：**

- 蜡烛图（Candlestick）- 当前
- 折线图（Line）- 只显示收盘价
- 面积图（Area）- 填充区域

### 4. 可配置指标

**MA 配置：**

- 可选择显示/隐藏
- 可配置周期（默认 5/10/30）
- 可配置颜色
- 根据时间周期自动调整（1m 用 5/10/30，1d 用 7/30/120）

### 5. 视觉优化

**工具栏：**

- 使用 Quasar 的分组按钮
- 添加图标
- 统一间距和对齐
- 深色主题优化

**图表：**

- 优化颜色对比度
- 改进网格线样式
- 更好的十字线效果
