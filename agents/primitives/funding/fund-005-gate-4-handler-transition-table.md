# FUND-005 Gate 4 Handler Transition Table

Type: Primitive
Audience: Coding assistants
Authority: High

## Purpose

Define handler-level transition tables for `FUND-005` Gate 4. This file consumes `agents/primitives/funding/fund-005-gate-3-target-validation-matrix.md` and removes implicit state-machine interpretation before later funding implementation iterations mutate protocol behavior.

## Rules

- Gate 4 is a transition-table artifact, not an implementation patch.
- Every row must cite one accepted Gate 3 row id.
- Every row must name exact target files and handlers.
- Every transition must state entry, chain, application, payload, required state, writes, emitted effects, next state, failure behavior, bounce behavior, claim effect, and tests.
- Do not use `webui/` as current frontend evidence.
- Use `webui-v2/` for current frontend and product-entry target work.
- Do not use generated GraphQL output under `webui-v2/src/__generated__/graphql/` as audited source.
- Do not touch Quasar framework files: `webui-v2/quasar.config.ts`, `webui-v2/src/boot/axios.ts`, `webui-v2/src/boot/i18n.ts`, `webui-v2/src/boot/notify-defaults.ts`, `webui-v2/src/boot/pinia.ts`.
- Operation entries use `Not applicable` in the bounce column.
- Tracked messages must define the exact bounce branch in the bounce column.
- Meme application handlers must not store initialize-liquidity route workflow state.
- `funding_correlation_id` is an opaque value allocated by swap and carried by meme. It is not a route id, pool id, token id, or semantic business key.
- `funding_origin_application` is runtime-derived from `runtime.authenticated_caller_id()` in swap operation handlers. It must not be accepted from operation payload.
- `SwapOperation::InitializeLiquidityFunded` and `SwapOperation::InitializeLiquidityFundFailed` must not include `funding_origin_application` in ABI.
- `SwapMessage::InitializeLiquidityFunded` and `SwapMessage::InitializeLiquidityFundFailed` carry `funding_origin_application` after the swap operation handler derives it from runtime authentication.
